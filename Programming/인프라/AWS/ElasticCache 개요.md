# Amazon ElastiCache 개요

> 정확한 제품명은 **ElastiCache**(Elasti + Cache)입니다. "ElasticCache"로 자주 오기되니 주의하세요.

Amazon ElastiCache는 완전 관리형 **인메모리 데이터 스토어** 서비스입니다. 데이터를 디스크가 아닌 메모리에 두기 때문에 매우 빠릅니다.

## 특징

- **1밀리초 미만**의 빠른 응답 시간을 제공합니다.
- 주로 **DB 앞단의 캐시**로 사용합니다. 자주 조회되는 데이터를 캐싱해 DB 부하를 줄이고 응답 성능을 개선합니다.
- 세션 스토어, 게임 리더보드, 실시간 분석처럼 **내구성(영속성)이 크게 중요하지 않은** 데이터 스토어로도 활용합니다.
- 애플리케이션에서 캐시를 조회/갱신하도록 **코드 변경이 필요**합니다.

## 지원 엔진

오픈소스 인메모리 솔루션 기반의 엔진을 선택할 수 있습니다.

| 엔진 | 특징 |
|---|---|
| **Redis / Valkey** | 스냅샷 백업, 복제, 고가용성(Multi-AZ), 다양한 자료구조 지원. 기능이 풍부해 대부분의 경우 선택 |
| **Memcached** | 구조가 단순하고 **멀티스레드**. 단순 키-값 캐싱에 적합 |

> **Valkey**: 2024년 Redis의 라이선스 변경 이후 등장한 Redis 호환 오픈소스 포크입니다. AWS ElastiCache는 Valkey를 지원하며, Redis 대비 비용이 저렴해 신규 구성 시 권장됩니다. (참고로 Redis는 전통적으로 싱글스레드로 명령을 처리합니다.)
